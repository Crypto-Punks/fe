#### Running the app


Lifecycle scripts included in crypto-punk-fe:

 * test -
    jest --verbose

 * start -
    webpack-dev-server --hot --mode development --devtool eval-source-map

available via `npm run-script`:

 * lint -
    eslint .

 * test:watch -
    npm run test -- --watchAll

 * build -
    webpack -p --devtool source-map
